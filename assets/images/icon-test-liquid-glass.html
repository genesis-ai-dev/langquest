Liquid Glass CSS

Light Mode:
.liquid-glass {
  backdrop-filter: blur(20px) saturate(180%);
  background: rgba(255, 255, 255, 0.25);
  border: 1px solid rgba(255, 255, 255, 0.3);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1), inset 0 1px 0 rgba(255, 255, 255, 0.4);
}
Dark Mode (Enhanced):
.liquid-glass-dark {
  backdrop-filter: blur(24px) saturate(200%) brightness(1.1);
  background: rgba(255, 255, 255, 0.15);
  border: 1px solid rgba(255, 255, 255, 0.25);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4), 0 0 0 1px rgba(255, 255, 255, 0.1), inset 0 1px 0 rgba(255, 255, 255, 0.3);
}
Dark Mode (Maximum Contrast):
.liquid-glass-dark-enhanced {
  backdrop-filter: blur(28px) saturate(220%) brightness(1.15) contrast(1.1);
  background: rgba(255, 255, 255, 0.12);
  border: 1px solid rgba(255, 255, 255, 0.35);
  box-shadow: 0 12px 40px rgba(0, 0, 0, 0.5), 0 0 0 1px rgba(255, 255, 255, 0.15), inset 0 2px 4px rgba(255, 255, 255, 0.2);
}
Icon Enhancement for Dark Backgrounds:
.bg-dark object, .bg-dark svg {
  filter: brightness(1.1) contrast(1.05);
}
